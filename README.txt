# Weather Bot

## Описание
Приложение Weather Bot - это бот для Telegram, который предоставляет информацию о погоде сделать в качестве теста для компании BobrAI.

## Перед запуском приложения

1. Убедитесь, что у вас установлен Docker.
2. Настройте PostgreSQL в Docker:
   - Измените файл `docker-compose.yml` для конфигурации базы данных.
   - Измените файл `application.yml`, чтобы он соответствовал вашей конфигурации Docker.
3. Через командную строку запустите контейнер Docker, команда для запуска:
    docker run -p 5432:5432 --name postgresql -e POSTGRES_PASSWORD=<пароль для входа в БД> -e POSTGRES_USER=<Имя пользователя> -e POSTGRES_DB=<Название БД> -d postgres:14
4. Создайте свою базу данных.

## Работа с ботом

1. /start - начало работы с ботом.
2. /weather <название города(на английском языке)> - получение погоды в указанном городе.
3. /setcity <название города(на английском языке)> - установка города по умолчанию.
4. Имеются кнопки управления, названия города указывать на английском языке.

## Логи
Логи приложения будут записываться в файл `logs/app.log`.

## Документация
Swagger автоматически сгенерирует документацию.
После запуска приложения вы сможете увидеть её по адресу: http://localhost:8080/swagger-ui.html.
